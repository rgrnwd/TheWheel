=======
## react-webpack-example

just an example using react, webpack and css modules...

## Dependencies

* **react** `0.14.2`
* **babel** `6.1.0`
* **webpack** `1.12.6`
* **webpack-dev-middleware** `2.0.0`
* **express** `4.13.3`
* **css skeleton** `2.0.4`
* **css modules**

## Run

```
npm install
npm start
open http://127.0.0.1:3000
```
